How to Join

1. Download MultiMC. Download the version you need for your system.
2. Once installed, open MultiMC and click on "Add Instance". Select ATLauncher and then find Sevtech Ages. Make sure the version is 3.2.3 in the bottom right, then select ok.
3.Once the instance is added, you'll need to log into your account. Click at the top right and select manage accounts. You can either add a Microsoft or Mojang account, depending on what you have. You can ask one of us for an account if you don't have one, we have a few extra shared.
4.Now you can launch Sevtech Ages. It will take a while to load, so go get a snack or check out some of the other games on this site. Once you are in, select multiplayer and add a server. The IP is 71.61.249.163. Now you can join the server and explore!


Trouble shooting

Failed to launch into loading screen:
Its likely using the wrong version of Java. You'll want to install Java 8. Once installed, select "Settings" at the top and click Java. Click auto-detect and it should give you options. You'll want to select version 1.8.0_xxx. Click ok and try launching again. (While you are here, you might want to check that the program has enough RAM)

Frequent Crashes:
Try giving the program more RAM. Select "Settings" from the top and select Java. How much RAM your computer has will limit how much you can allocate. My minimum is set to 8192 MiB and I haven't had any issues. Most computers have at least 16GB of RAM so this should be fine.
